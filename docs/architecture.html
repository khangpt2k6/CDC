Observability plane cross-cutting
🔥 Prometheus
Scrapes the Go worker /metrics endpoint on an interval.

📟 Grafana (metrics)
Same Grafana, second datasource: pipeline health next to the analytics view.
Flow: Go worker → Prometheus (pull /metrics) → Grafana. This plane observes the processing plane; it is not in the data path, so a metrics outage never blocks ingestion.

Correctness: Kafka delivery is at-least-once, but the worker commits offsets only after ClickHouse acks the flush, and ReplacingMergeTree collapses any replayed duplicate on _version - so the end-to-end result is exactly-once in effect.
